Welcome to the Grave Site

The Grave Site

As you develop the grave site you will learn the basics of coding a website. The course will cover many different tags but no one can remember them. IF you need to look up or regerence a tag or just remember what one does, go look around w3schools.com

Web Development Essentials

In Web Essentials we will learn how to code the basic structure of the page using HTML5 as well as style the page using CSS3.

Web Databases

In Web Databases you will learn how to setup and structure a Database using MySql.

Server Side Scripting

In server side scripting we will be teaching you about PHP. PHP allows you to make a web site dynamic, as well as interact with, retrieve, and post data to the Databsae.

Client Side Scripting

Client Side Scripting will cover JavaScript & JQuery adding functionality to the website and completing the basic languages used in Web Development.
By the end of this class you will have completed most of the core classes in the program.
